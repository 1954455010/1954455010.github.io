发表于 2023-10-29 更新于 2023-11-10

进入for循环

①首先执行表达式1，表达式1是也是初始化语句。

②接着执行表达式2，如果表达式2的条件是真（或说表达式2条件成立）就执行for循环内部的语句块，语句块执行完成后自动执行表达式3（一般是是自增或自减）

③表达式3执行完成后，返回判断表达式2的条件是否成立。如果是真继续第②步的操作，如果不成立则跳出循环体。for循环如果不带{}那么会执行最近的语句块。

发表于 2023-10-27

（1. if判断语句

if 语句：一个 if 语句 由一个布尔表达式后跟一个或多个语句组成。

（2.关系运算符

<1>关系运算符的作用是判断符号两边大小的关系

<2>关系运算符的两边可以是变量、数值 或 表达式

变量

数值

表达式

<3>当两个值的关系符合关系运算符的预期时,关系运算的结果为整数1，否则为整数0

<4>运算符嵌套

  • 关系运算符是允许嵌套使用的，运算的结果可以继续作为关系运算符的一个参数

  • 同优先级的关系运算符结合律是从左往右的。

因为2 > 3的结果为1，所以2 > 3 > 0等价于1 > 0，输出的结果为：1

因为3 > 2的结果为1，所以3 > 2 > 1等价于1 > 1，输出的结果为：0

<5>优先级

所有的关系运算符的优先级比算术运算的低，但是比赋值运算的高

算术符的优先级

  • 高优先级： <、 <=、>、>=

  • 低优先级： ==、!=

C语言学习-3 (交换变量和复合赋值及递增递减)

发表于 2023-10-25 更新于 2023-11-09

（1.交换变量

交换a,b两个变量的值

程序表达的是顺序执行的动作，而不是关系。

是依次执行的，结果使得a和b都得到b原来的值

（2.复合赋值

5个算术运算符，＋-/%，可以和赋值运算符“=”结合起来，形成成复合赋值运算符
“+=”、”-=”、“*=”、“/=”和“%=”

注意 两个运算符中间不要有空格

要把赋值右边的算完再进行左边的赋值运算

（3.递增递减

<1>“＋＋”和“–”是两个很特殊的运算符，它们是单目运算符，这个算子还必须是变量。这两个运算符分别叫做递增和递减运算符，他们的作用就是给这个变量+l或者-l。

<2>++和–可以放在变量的前面，叫做前缀形式，也可以放在变量的后面，叫做后缀形式。
a++的值是a加I以前的值，而++a的值是加了I以后的值，无论哪个，a自己的值都加了l了。

（4.输入函数 scanf

  1. 使用”%f”格式控制符来读取浮点数。例如，scanf(“%f”, &num)
    语句将会从标准输入中读取一个浮点数，并将其存储在变量num
    中。
  2. 在读取浮点数时，可以输入整数或小数，并且可以使用科学计数
    法表示。例如，输入”3.14”、”2.0”或”1.23e-4”都是合法的浮点数输
    入。
  3. 在读取浮点数时，可以使用scanf函数的其他格式控制符来控制
    输入的格式。例如，”%lf”表示读取一个双精度浮点数，”%Lf”表示
    读取一个长双精度浮点数。
  4. 在使用scanf函数读取浮点数时，需要注意输入数据的合法性。
    如果输入的数据不是一个合法的浮点数，scanf函数会返回0，并且
    输入缓冲区中的数据不会被改变。因此，为了确保输入的数据合法，
    可以在使用scanf函数前进行数据校验。
